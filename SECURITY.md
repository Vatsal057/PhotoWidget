# Security Policy

## Supported versions

Only the latest release is supported with security fixes.

## Reporting a vulnerability

Please report security issues privately, not as public GitHub issues:

- Use GitHub's private reporting: **Security tab → Report a vulnerability**, or
- Email **kvaghasiya057@gmail.com** with details and steps to reproduce.

You'll get a response within a few days. Once a fix ships, the issue can be
disclosed publicly.

## Scope notes

PhotoWidget stores photos locally in an App Group container and makes no
network requests. Reports about the build/signing scripts (`scripts/`) are
in scope too.
